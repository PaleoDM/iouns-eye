## Description

Cloudpierce Spire is a gleaming high-district landmark in Rifthaven that serves as the launchpoint for the legendary Roc Run airship race. The spire features security checkpoints that the party had to talk their way through before boarding Captain Lysa Vex's airship, the Brass Wing. From Cloudpierce, competing airship crews race to the Southern Roc Roost to secure an unfertilized roc egg and return it to the Harrowside Docks within 16 hours.
